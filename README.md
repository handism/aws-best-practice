# aws-best-practice

AWS のベストプラクティスを Infrastructure as Code (IaC) で管理するためのリポジトリです。

## ディレクトリ構成

- `iac/`: IaC ファイルを配置するディレクトリ

## 使い方

1. `iac/` 配下に Terraform や CloudFormation などの定義を追加します。
2. 必要に応じて検証・デプロイ手順を本 README に追記します。

## 今後の拡張例

- コーディング規約の追加
- Lint / Format / Validate の自動化
- CI/CD パイプラインの整備
